## User
@.cursorrules 내용을 기준으로 .cursor/rules 아래에 규칙 파일 5개를 설계해줘.  
지금은 파일 생성/수정은 하지 말고, 각 파일의 최종 초안 텍스트만 보여줘.

대상 파일:
- magicsquare-project.mdc
- magicsquare-ecb-architecture.mdc
- magicsquare-forbidden.mdc
- magicsquare-python-code-style.mdc
- magicsquare-tdd-testing.mdc

요구사항:
1) 각 파일은 .mdc 형식으로 작성하고 YAML frontmatter 포함:
- description
- globs (필요한 경우만)
- alwaysApply
2) 규칙은 한국어로 간결하게, 파일당 30~50줄 내외
3) 1파일 1책임 원칙 유지, 중복 문장 최소화
4) 기존 .cursorrules 의미를 보존하되, 지정된 항목으로 분리
5) 각 파일 마지막에 "적용 예시(짧게)" 2줄 추가
6) 출력 형식:
- 먼저 파일별 역할 요약 1줄씩
- 그 다음 각 파일의 전체 .mdc 내용(코드블록)
